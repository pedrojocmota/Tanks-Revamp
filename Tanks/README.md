# 🚀 Tanks-Revamp - Unity Multiplayer Game

A 3D battle tank game created using Unity, based on the official Unity Learn course:  
**["Tanks! - Make a Battle Game for Web and Mobile"](https://learn.unity.com/course/tanks-make-a-battle-game-for-web-and-mobile)**

## 🎮 About the Project

This project is a third-person battle tank game featuring:
- Single-player and multiplayer gameplay
- Dynamic arena combat
- Power-ups and damage system
- Mobile and web compatibility

The game serves both as a learning tool and a base for further gameplay experimentation.

## 🛠️ Built With

- [Unity](https://unity.com/) 2021.3 LTS or later
- C#
- Unity Networking (UNet or replacement depending on Unity version)

## 📦 Features

- Third-person tank controller with physics
- Aiming and shooting mechanics
- Health system with visual feedback
- Respawning and match timer
- Mobile and WebGL build support

## 🧪 How to Run

1. Clone the repository:
   ```
   git clone https://github.com/pedrojocmota/Tanks-Revamp.git
   cd Tanks-Revamp
   ```
2. Open the project in Unity Hub.

3. Select the scene Arena.unity or MainMenu.unity.

4. Click Play to test in the Editor.

5. Build for WebGL or Android/iOS as needed.
